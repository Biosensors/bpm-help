What is Biosensors Process Management (BPM)?

Biosensors Process Management (BPM) is a system developed in-house to help bring paper processes online and streamline business processes for optimal efficiency.

The system allows user to create electronic form with a predefined approver/action taker/acknowledger workflow.

How to login to BPM?

No account creation needed as BPM is integrated with Active Directory. Just login using your windows ID and password and you can start to build template or initiate process using the released template.

  1. Click on the "LOGIN" button.
  2. Enter your windows user ID. (e.g. ms.tan)
  3. Enter your windows login password.
  4. Select your company domain.
  5. Click on "SUBMIT" button.
  1. You will see your name display upon successful login.
  2. To logout, simply click on your name, followed by another click on the "Logout" button in the profile dialog.


Is BPM mobile friendly?

BPM is a mobile friendly system and it streamlined experience across different devices. You could access and use the system either through your computer or mobile devices (i.e iOS or Android devices).

Mobile Phone View

Is BPM a multi-language system?

Yes. BPM currently supports multiple languages such as English, Simple Chinese, French and Japanese. New language can be added based on business needs.

What’s new in upgraded BPM?

  1. Access BPM via public network without VPN.
  2. New field control For E-Signature.

  3. New field control for table listing (aggregation and formula supported).

  4. Integration capability with other system (e.g. SAP/Biotrack/Eclaim) via data configuration control.
  5. New template setting for parallel approval/action taken.
  6. New template setting for group user action. (i.e. require only one of the user in group for action)
  7. Save as favorite for frequently used template.
